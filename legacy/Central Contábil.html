Conciliação Bancária Assistida

Anexe o extrato do banco e o relatório de contas pagas/recebidas enviado pelo cliente. O confronto por valor, data e descrição é automático — e você pode acionar a IA para desempatar os casos duvidosos antes de exportar a planilha final.

01
Censurar dados sensíveis

Antes de anexar, tarje ou remova informações sensíveis do cliente e da Navecon nos arquivos originais.

Revise o extrato bancário e o relatório de contas antes de subir aqui: apague ou tarje CNPJ, número de conta/agência completos, saldos de outras contas e qualquer dado de terceiros que não seja necessário para a conciliação. Mantenha visíveis apenas as colunas essenciais — data, descrição/histórico e valor.

Abrir editor de PDF (iLovePDF) ↗

Use essa ferramenta para tarjar os dados no PDF antes de anexar na etapa abaixo.

02
Anexar arquivos

Formatos aceitos: .xlsx, .xls, .csv, .pdf

B

Extrato bancário do cliente

Colunas típicas: data, histórico/descrição, valor (ou débito/crédito). PDF também é lido, uma linha por lançamento.

C

Relatório de contas pagas/recebidas

Arquivo enviado pelo cliente. Colunas típicas: data, histórico, valor, tipo. PDF também é lido, uma linha por lançamento.

03
Confirmar colunas

A IA analisa o cabeçalho e a amostra do arquivo e já sugere o mapeamento — confira antes de rodar a conciliação.

Diferença de centavos aceitável entre o valor do banco e o valor das contas para ainda considerar "o mesmo valor" (ex: arredondamento). Aumente só se seus arquivos costumam ter pequenas diferenças de centavos.

Quantos dias de diferença entre a data do banco e a data das contas ainda são aceitos (compensação de boleto, feriado, fuso de processamento). Quanto maior, mais candidatos entram na comparação — e mais chance de confundir lançamentos parecidos.

Quando há mais de um lançamento com valor/data parecidos, o sistema compara as descrições e só concilia sozinho se a semelhança do texto bater esse percentual (0 a 1). Abaixo disso, o item vai para "Sugestões" para você confirmar. Suba o número para exigir mais certeza; abaixe para automatizar mais.

Nada é enviado para fora do navegador nesta etapa.
04
Resultado da conciliação

Confira os casos automáticos, refine as sugestões com IA e exporte.

Calculadora de Estoque

Use quando o cliente não envia controle de estoque. A partir do estoque inicial, das compras e do faturamento do período, a ferramenta estima o estoque final aplicando um percentual de custo sobre a receita.

01
Dados do período

Preencha após a importação das notas no fiscal. Valores em R$.

1º cálculo — Disponível para venda (X)

X = EI + Compras. Pegue as informações no módulo contábil, depois da importação.

2º cálculo — Faturamento do período (Y)

Y = Total Vendas + Total Prestação de Serviços.

3º cálculo: A = Y × %. Essa porcentagem pode variar conforme o ramo da empresa — ajuste conforme o histórico do cliente.

02
Estoque final estimado

4º cálculo: EF = X − A. Atualiza automaticamente conforme os dados acima.

Análise Tributária Comparativa

Simulação comparativa entre Simples Nacional, Lucro Presumido e Lucro Real para grupos econômicos multi-empresa, com relatório e dashboard prontos para apresentar ao cliente.

Nenhum grupo selecionado

Criar Novo Grupo

Cada grupo econômico/cliente é salvo separadamente. Você pode voltar a ele a qualquer momento nesta lista.

Grupos Já Cadastrados

Selecione um grupo para continuar a análise de onde parou, ou exclua um cliente que deixou de ser atendido.

Dados do Grupo

Identificação da análise. O app trabalha com N empresas de um mesmo grupo econômico, permitindo intercompany entre elas.

Empresas do Grupo

Cadastre cada empresa. Todas as empresas terão seus tributos calculados nos 4 cenários (Presumido sem/com Simples e Real sem/com Simples), independente do regime atual.

Como o motor de cálculo apura cada tributo

  • Simples Nacional: valor mensal apurado informado diretamente (aba "Apuração do Simples") — não recalculado internamente por Anexo.
  • Presunção IRPJ/CSLL (Presumido): Receita de Comércio sempre 8%/12% (fixo); Receita de Serviço + Intercompany conforme a atividade selecionada por empresa (Serviços em geral 32%/32%, entre outras).
  • Receita mensal: dividida em Comércio, Serviço e Intercompany — necessário para empresas com atividades mistas, já que a presunção de IRPJ/CSLL difere entre elas.
  • IRPJ: 15% sobre a base + adicional de 10% sobre o que exceder R$ 20.000/mês (acumulado no ano). CSLL: 9%.
  • PIS/COFINS Presumido (cumulativo): 0,65% + 3,00% sobre a receita bruta.
  • PIS/COFINS Real: input manual de débito/crédito mensal (aba "Impostos").
  • INSS Patronal: cálculo detalhado (Folha/GPS) para empresas do Simples; valor apurado informado manualmente para Presumido/Real.
  • IPI, ICMS (inclusive TTD — guia própria), ISS, Fundos: apurados por fora, informados na aba "Impostos" (apenas Presumido/Real).
  • CBS (Reforma Tributária): projeção de transição a 9,45% sobre o faturamento — validar periodicamente.
  • Dados mensais: preenchidos por ano-calendário (2021 em diante), permitindo comparativos de evolução anual.

Dados Mensais por Empresa

Preencha os valores mês a mês. Use o botão → em cada linha para replicar o valor de Janeiro para todos os meses. As abas variam conforme o regime atual de cada empresa.
Cada ano é salvo separadamente — trocar o ano não apaga os dados já preenchidos em outro ano.
